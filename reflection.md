# Reflection Praktikum DevOps — Pertemuan 01

Melalui praktikum ini, saya mulai memahami pentingnya menyiapkan development environment yang terstandarisasi dalam praktik DevOps. Proses instalasi Git, Docker, dan Visual Studio Code memberikan gambaran awal tentang tools yang akan sering digunakan dalam workflow DevOps.

Harapan saya dari praktikum ini adalah dapat memahami implementasi DevOps secara lebih mendalam, khususnya dalam hal automation, containerization, dan continuous integration/continuous deployment (CI/CD). Saya juga berharap mampu mengaplikasikan konsep DevOps dalam proyek nyata sehingga dapat meningkatkan keterampilan sebagai calon praktisi informatika.
